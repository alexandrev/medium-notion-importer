## 🏁 <span id="how_to_use">How to use </span>

You need to define the .env file with the following data:

```
NOTION_KEY=secret_XXXXXXX
NOTION_DATABASE_ID=YYYYYYYYYYYY
MEDIUM_USER_ID=medium_user
SCHEDULER_CRON=* * * * * * 
```


Launching the tool

```
npm install
node index.js
```
